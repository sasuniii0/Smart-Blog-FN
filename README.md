# Smart Blog Frontend

Smart Blog Frontend is a modern, responsive blogging platform UI built using React, designed for smooth reading, clean layouts, and seamless user interaction.

## Tech Stack

1. React
2. Vite
3. TailwindCSS
4. Axios

## Run the project
```
npm install
npm run dev
```

## Folder Structure

```
src/
 ├── components/
 ├── pages/
 ├── hooks/
 ├── utils/

```
## API Connection

The frontend connects to the Smart Blog Backend using environment variables:

Create a .env file:

```VITE_API_URL=http://localhost:5000/api/v1```
